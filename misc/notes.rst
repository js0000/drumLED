modes
=====
- 16 modes divided into 4 groups of 4 (details in modes.csv)
- hexidecimal ids
- 0-3 single color
- 4-7 multi color
- 8-0xB mic
- 0xC-0xF use programming logic


hardware
========
- **display** for mode id
- **button** to change mode
- **pot** to change something
- **microphone** for audio input
