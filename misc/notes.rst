MODES
=====
- 16 modes divided into 4 groups of 4
- hexidecimal ids
- 0-3 single color
- 4-7 multi color
- 8-0xB mic
- 0xC-0xF use programming logic


HARDWARE
========
- **display** for mode id
- **button** to change mode
- **pot** to change something
- **microphone** for audio input
